# Readme

## Wochenenden-Planer
Eine Internetseite wo meine Freunde und ich ein gemeinsames Wochenende planen. 
Die Leute können sich eintragen wer Zeit hat und einen Vorschlag machen, was man machen könnte. Features die ich plane zum einbauen, ist ein Login, mit dem sich berechtigte Leute mit den Zugangsdaten auf die Internetseite einloggen können. Die Startseite sollte ein kleines Intro sein mit Bildern, die zeigen was für Erlebnisse man schon gemeinsam hatte, gleichzeitig soll es kurz erklären für was die Seite ist. Die zweite Seite soll ein interaktiver Kalender sein, wo man Datum, Zeit und Vorhaben eintragen kann.
